初期セットアップガイド

LINEチーム — 業務開始前にやっておくこと（20分で終わります）

LINEチームでは Claude Code（AIアシスタント）を中心に業務を行います。
この手順に沿って環境を整えてください。

わからないことがあったら Ren に聞いてください。 全部フォローします。

1 Claude Code をインストール 必須 5分

デスクトップ版とVS Code版、両方インストールしてください。
普段はVS Code版をメインで使いますが、デスクトップ版も入れておくと便利です。

⚠ Windowsの方は先にこれをやってください

Claude Code は Git Bash が必須です。

  1. git-scm.com/downloads/win から Git for Windows をダウンロード
  2. インストール時に「Add to PATH」にチェックを入れる
  3. インストール完了後、VS Code を再起動

インストール済みなのにエラーが出る場合：

PATHに通ってない可能性あり。Windowsの「システム環境変数の編集」→「環境変数」→「新規」で以下を追加：

CLAUDE_CODE_GIT_BASH_PATH = C:\Program Files\Git\bin\bash.exe

設定後、VS Code を再起動。

VS Code版（おすすめ）
デスクトップ版

前提：VS Code がインストール済みであること
まだの場合 → VS Code ダウンロード

  1. VS Code を開く
  2. キーボードで Ctrl + Shift + X を押す

    → 左に「拡張機能」パネルが開く

  3. 上の検索バーに Claude Code と入力
  4. 「Claude Code」（発行元: Anthropic、認証済みマーク付き）を見つける

    → 似た名前の拡張がありますが、Anthropic公式のものを選んでください

  5. 「インストール」をクリック
  6. インストール完了後、VS Code を再起動

    → 再起動後、右上に ✱ アイコンが表示されればOK

  7. ✱ アイコンをクリック → ログイン画面が出る

✱ アイコンが見つからない場合：
Ctrl + Shift + P → 「Claude Code」と入力 → 候補から選択

  1. claude.ai/download にアクセス
  2. お使いのOS用のインストーラーをダウンロード
  3. ダウンロードされたファイルを実行
  4. インストーラーの指示に従ってインストール
  5. インストール完了後、アプリを起動
  6. ログイン画面が表示される → 次のステップ2へ

「WindowsによってPCが保護されました」と出た場合：
「詳細情報」→「実行」をクリックすれば進めます。

2 アカウントでログイン 必須 1分

Claude Codeのログイン情報は 別途Renから共有します。

ログイン方法はVS Code版・デスクトップ版ともに同じ：

  1. ログイン画面で「Continue with email」または「Log in」をクリック
  2. 共有されたアカウント情報を入力
  3. ログイン完了 → Claude Codeが使える状態になる
3 GitHubアカウントを作成 必須 5分

チームのデータ共有に GitHub（エンジニア向けのクラウドストレージのようなもの）を使います。
無料アカウントでOKです。

  1. github.com/signup にアクセス
  2. メールアドレス → パスワード → ユーザー名を入力
  3. メール認証（届いたメールのリンクをクリック）
  4. プランは Free を選択

ユーザー名のコツ： 本名でもニックネームでもOK。短くて覚えやすいものがベスト。後から変更もできる。

作成したら、ユーザー名を Ren に伝えてください。
チームのリポジトリへの招待に必要です。

4 Git の初期設定 必須 2分

ターミナル（VS Codeの場合は下部のターミナルパネル）で以下を実行します。

ターミナルの開き方（VS Code）：
メニュー「ターミナル」→「新しいターミナル」、または Ctrl + `

① Gitが入っているか確認

git --version

バージョン番号が表示されればOK。エラーが出たらRenに聞いてください。

② 名前とメールアドレスを設定

git config --global user.name "自分の名前"
git config --global user.email "GitHubに登録したメールアドレス"

この設定は初回だけ。一度やれば次からは不要です。

5 作業リポジトリをダウンロード 必須 2分

Renから招待メールが届いたら、メール内の「View invitation」→「Accept invitation」で承認してください。

メールが届かない場合：
Renに連絡してください。直接リンクで招待します。

承認後、ターミナルで以下を実行：

cd ~/Desktop
git clone https://github.com/Ren-japan/star-team.git
cd star-team

デスクトップに star-team フォルダができます。

② 自分の名前を登録する

以下の Noah を自分の名前に書き換えて実行：

echo 'export STAR_TEAM_USER="Noah"' >> ~/.zshrc
source ~/.zshrc

Noah の部分を自分の名前に変えてください（Kate / Rinon）。
Windowsの場合は ~/.zshrc の代わりに ~/.bashrc を使ってください。

6 Chrome拡張「Claude in Chrome」を入れる おすすめ 2分

Claude CodeがChromeブラウザを操作できるようになる拡張機能です。

  1. Chromeで Claude in Chrome を開く
  2. 「Chromeに追加」をクリック
  3. 追加後、拡張機能アイコンをクリックして接続を確認

なぜ入れるの？

これがあると、Claude CodeがWebページを直接見に行ける。

  • 「このURLの記事チェックして」→ 実際にページを開いて読む
  • 「競合サイトの構成を調べて」→ ブラウザで開いて抽出

ないとURLの中身が読めない。あると世界が変わります。

7 動作確認 必須 1分

全部うまくいったか確認します。ターミナルで以下を実行：

cd ~/Desktop/star-team
python3 manage.py list

タスク一覧が表示されたらセットアップ完了です！

おつかれさまでした。 これで業務を始める準備が整いました。
具体的な業務の進め方は、初日にRenから説明します。

エラーが出た場合：

  • python3: command not found → Renに連絡してください
  • No such file or directory → cd ~/Desktop/star-team を先に実行してから再度試す
  • その他のエラー → スクリーンショットを撮ってRenに送ってください

セットアップチェックリスト

  • Claude Code をインストールした（VS Code版 + デスクトップ版）
  • Claude Code にログインできた
  • GitHubアカウントを作成した
  • ユーザー名をRenに伝えた
  • 招待メールを承認した
  • Git の初期設定をした（名前 + メール）
  • リポジトリを clone した
  • 自分の名前を登録した（STAR_TEAM_USER）
  • python3 manage.py list でタスクが表示された
  • （おすすめ）Chrome拡張を入れた

ステップ1〜5が終わっていればOKです。
ステップ6は余裕があればやっておくと便利です。
業務の進め方は初日にRenから説明します。
